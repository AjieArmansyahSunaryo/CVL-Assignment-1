# Image Enhancement Implementation

Repository ini berisi implementasi beberapa teknik **image enhancement** menggunakan Python dan OpenCV.

Detail implementasi dapat dilihat langsung pada file notebook:
`CVL_Assignment01.ipynb`

## Implementasi

Proyek ini mengimplementasikan tiga metode peningkatan kualitas citra:

1. **Gamma Correction**
   Mengubah gambar yang gelap menjadi lebih terang.

2. **Image Sharpening (High-Pass Filter)**
   Menajamkan gambar dengan menonjolkan edge atau perubahan intensitas.

3. **Contrast Stretching**
   Meningkatkan kontras pada gambar dengan memperluas rentang nilai intensitas piksel.

## Output

Setiap metode menghasilkan perbandingan antara:

* **Original Image**
* **Processed Image**

Visualisasi hasil ditampilkan menggunakan **Matplotlib** sehingga perubahan pada gambar dapat terlihat dengan jelas.
